Calculator Instructions

How to Use the Workforce Calculator

Step 1: Enter Growth Factors

  • Select values for each management level using the dropdowns
  • Values will automatically update the Compound Impact Factor
  • Ensure values are within the specified ranges

Step 2: Enter Current Headcount

  • Input the current headcount for each level
  • Use whole numbers only
  • All fields must be filled

Step 3: Calculate Forecast

  • Click the 'Calculate Forecast' button
  • Review the calculated projections
  • Make adjustments if needed and recalculate

Step 4: Export Results

  • Export buttons will be enabled after calculation
  • Excel export includes detailed worksheets
  • PDF export provides a formatted report

Understanding Growth Factors

Economic Factor (-0.2 to +0.2)

  • Positive: Growing economy
  • Zero: Stable economy
  • Negative: Economic downturn

AI/Automation Factor (-0.3 to +0.1)

  • Typically negative due to automation
  • Can be positive if AI creates more jobs

Work Transformation (-0.2 to +0.2)

  • Positive: Digital transformation creating jobs
  • Negative: Consolidation reducing positions

Retention Factor (0.7 to 1.2)

  • Below 1: High turnover
  • Above 1: Strong retention
  • 1.0: Normal retention

Skills Factor (0.8 to 1.3)

  • Below 1: Skills Gap limiting Growth
  • Above 1: Strong skills availability
  • 1.0: Balanced skills market
